## b_tree
递归三步：
1. 递归入参和出参
2. 递归出口
3. 当前层逻辑

递归和回溯紧挨着
递归中全局变量的作用
递归中返回值的好处

递归是否需要返回值：
- 搜索整棵二叉树 && 不用处理递归返回值，递归函数就不要返回值
- 搜索整棵二叉树 && 需要处理递归返回值，递归函数就需要返回值
- 搜索其中一条符合条件的路径，那么递归一定需要返回值，因为遇到符合条件的路径了就要及时返回
